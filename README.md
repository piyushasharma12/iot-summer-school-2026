# IoT Summer School 2026

## About the Repository

This repository contains all the work I completed during the **IoT Summer School 2026 at IIT Jammu**. It includes weekly assignments, Arduino and ESP32 programs, circuit simulations, and IoT projects that I worked on throughout the course.

I have organized the repository week-wise so that it is easy to follow my learning and the projects completed during each week.
