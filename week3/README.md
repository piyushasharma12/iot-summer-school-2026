# Week 3 – IoT Summer School 2026

## Overview

This folder contains all the work completed during **Week 3** of the **IoT Robotics & Drones Summer School 2026** at IIT Jammu.

The week's activities focused on learning ESP32-based IoT applications, sensor interfacing, automation, wireless communication concepts, and solving the practical problems provided in **Assignment 3**.
