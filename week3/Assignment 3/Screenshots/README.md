
# Week 3 - IoT Summer School 2026

## Overview

This folder contains the work completed during **Week 3 of the IoT Robotics & Drones Summer School 2026** at IIT Jammu.

During this week, I worked on different IoT concepts including sensor interfacing, ESP32 programming, IoT communication, automation systems, and practical simulations using Wokwi.

The assignments focused on understanding how sensors collect real-world data, how microcontrollers process this data, and how IoT systems can be designed for different applications.
