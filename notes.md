# Branch Plan

## dev
This branch is used to develop, test, and integrate new features before merging them into the main branch.

## sensor-module
This branch will contain all sensor-related programs and experiments, including DHT11, LDR, PIR, Ultrasonic Sensor, Servo Motor, and Relay projects.

## wireless-module
This branch will contain all wireless communication projects, including Bluetooth, BLE, Wi-Fi, MQTT, and HTTP-based IoT applications using ESP32/ESP8266.
